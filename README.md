# WearAll

An AI-powered smart styling app and virtual wardrobe manager.

## What it does

- Manages a user's digital wardrobe
- Provides outfit recommendations via an AI virtual stylist named Mia
- Factors in weather, calendar events, and personal wardrobe data to suggest what to wear

## Tech Stack

- **Frontend:** React
- **Backend:** Node.js
